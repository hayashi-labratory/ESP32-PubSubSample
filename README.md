# ESP32 PubSub Sample

## Description
学生実験で使用するESP32マイコンのサンプルプログラム．  
基本的には *esp_xxx_main.ino* の中身を書き換えます．
細かな設定や処理については peripheral.ino/.hに分離して隠しています．

## Folder description
- **exp0_sample**  
ESP32マイコンに慣れる用のサンプルプログラム．
- **exp1_sample**  
実験1で使用する Publish/Subscribe サンプルプログラム．
- **exp2_sample**  
実験2で使用するサンプルプログラム．
